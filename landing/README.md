# Fluidic Landing Page

A premium, futuristic single-page website for Fluidic — the post-blockchain Wave-Field Mesh network.

## Stack

- Next.js 16 (App Router)
- React + TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Sections

- **Hero** — Shimmer headline, animated wave-field mesh, CLI snippet with copy button
- **Architecture Comparison** — Blockchain vs. Fluidic Mesh cards
- **Key Features** — Glowing interactive grid
- **Developer Sandbox** — Dark syntax-highlighted code editor mockup
- **Footer CTA** — Ecosystem links and dual call-to-action buttons

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

Static export is written to `./dist`.
